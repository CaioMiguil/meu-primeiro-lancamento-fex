## Variável de ambiente na Vercel

Antes de testar o envio para o Google Sheets, configure na Vercel:

Nome: GOOGLE_SHEETS_URL
Valor: sua URL do Google Apps Script Web App

Depois faça o deploy/redeploy do projeto.
